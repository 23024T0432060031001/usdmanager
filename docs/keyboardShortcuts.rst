Hidden shortcuts
----------------

For ease of use, there are some extra shortcuts not shown in the menus themselves:

.. list-table::
   :header-rows: 1

   * - Command
     - Shortcut
   * - Back
     - Backspace
   * - Find
     - Ctrl+F
   * - Zoom In
     - Ctrl+=
   * - Next Tab
     - Ctrl+Tab
   * - Previous Tab
     - Ctrl+Shift+Tab
   * - Reload
     - F5
   * - Indent (if text is selected)
     - Tab
   * - Unindent (if text is selected)
     - Shift+Tab
